# ✈️ Flight Delay Exploratory Data Analysis (My First EDA Project)

## 📌 About This Project

Welcome to my **first Exploratory Data Analysis (EDA) project!**

As I continue my journey toward becoming an AI Engineer, I wanted to apply the Python, Pandas, NumPy, Matplotlib, and Seaborn skills I've been learning to a real-world dataset.

In this project, I explored a flight delay dataset to understand its structure, clean the data, visualize important patterns, and practice extracting meaningful insights from data.

Although this is my first EDA project, it represents an important milestone in my data science learning journey.

---

## 🎯 Project Objectives

* Understand the dataset and its features
* Perform basic data cleaning
* Explore numerical and categorical variables
* Detect missing values, duplicates, and outliers
* Create meaningful visualizations
* Practice interpreting data through business-focused insights

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 What I Explored

### Data Understanding

* Dataset overview
* Shape and dimensions
* Data types
* Summary statistics
* Missing value analysis
* Duplicate value check
* Unique values

### Univariate Analysis

* Distribution of departure delays
* Flight distance distribution
* Outlier detection using boxplots
* Airline distribution
* Origin airport distribution
* Cancelled flights
* Diverted flights

### Bivariate Analysis

* Departure Delay vs Arrival Delay
* Distance vs Departure Delay
* Airline vs Departure Delay
* Origin Airport vs Departure Delay
* Day of Week vs Departure Delay
* Correlation Heatmap
